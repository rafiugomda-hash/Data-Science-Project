# BCS 404: Introduction to Data Science Project
Author: Gomda Rafiu Chelti
Institution: Accra Technical University

## Project Overview
This repository contains my final project for BCS 404. It demonstrates practical competence in applying Python libraries for data visualisation, statistical analysis, and machine learning using the real-world Titanic dataset.

## Files in this Repository
* titanic.csv: The dataset used for analysis.
* titanic_analysis.ipynb: The Jupyter Notebook containing all Python code, data cleaning, visualisations, and the Logistic Regression model.
* Project_Report.pdf: The final written report discussing methodologies, statistical insights, and machine learning results.

## Technologies Used
* Python
* Pandas & NumPy
* Matplotlib & Seaborn
* Scikit-Learn
